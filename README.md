## Usage

This project is created by IDEA with Maven, so you know how 2 deploy that right :)

###### Before deploying

1. JDK 1.8.0 PLEASE!!!
2. Tomcat 8.5

###### Go!

```sh
git clone https://github.com/anzai249/CinemaManager.git
```

Use IntelliJ IDEA to edit or debug.
